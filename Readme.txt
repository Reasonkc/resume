===========================================
  Rijan Khatri - Personal Portfolio Website
===========================================

Welcome to my personal portfolio website repository!

ABOUT
-----
This is a responsive personal portfolio website showcasing my skills, projects, and experience as a Computer Engineering student, Frontend Developer, and UX/UI Designer.

FEATURES
--------
- Responsive design that works on all devices
- Clean and modern UI/UX
- Portfolio section showcasing my projects
- Skills visualization with progress bars
- Contact form for getting in touch
- Smooth scrolling navigation
- AOS (Animate On Scroll) animations

TECHNOLOGIES USED
-----------------
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- AOS (Animate On Scroll)
- GLightbox
- Isotope Layout
- Swiper
- Typed.js

PROJECT STRUCTURE
-----------------
resume/
├── index.html                          # Main homepage
├── portfolio-details-carrom.html       # Carrom Board project details
├── portfolio-details-resume.html       # Resume website project details
├── portfolio-details-scimatch.html     # SciMatch project details
├── assets/
│   ├── css/                           # Stylesheets
│   ├── img/                           # Images and portfolio screenshots
│   ├── js/                            # JavaScript files
│   └── vendor/                        # Third-party libraries
└── forms/                             # Contact form handling

HOW TO USE
----------
1. Clone this repository
2. Open index.html in your web browser
3. Customize the content with your own information
4. Update images in assets/img/ folder
5. Deploy to your hosting service

CUSTOMIZATION
-------------
To personalize this website:
- Update personal information in index.html
- Replace portfolio images in assets/img/portfolio/
- Update social media links
- Modify the color scheme in assets/css/style.css
- Add your own projects in the portfolio section
